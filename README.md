### Setup

1. Clone the repo (HTTPS or SSH)

```sh
https://github.com/k-henningson/Evogym.git
```

```sh
git@github.com:k-henningson/Evogym.git
```

2. Install dependencies

```sh
npm install
```

3. Start

```sh
npm run dev
```

4. Run local host link from the terminal in your browser
